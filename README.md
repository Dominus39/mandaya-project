# mandaya-booking-system-project

## Description: 

web aplication for Booking system hotels like order, pay, cancel, dan topup. Split into 3 micro services:
- user-service: process for user needs.
- order-service: process searching and book rooms.
- payment-service: process payment with 3rd party Xendit
